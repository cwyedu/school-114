# 🌐 Lab4：清單與表格頁面

---

## 🎯 本次學習目標

本次將學習如何用 HTML 建立清單與表格：

1. 學會使用 **清單標籤（ul、ol、li）**
2. 學會使用 **表格標籤（table、tr、td、th）**
3. 建立 **lab4/index.html 頁面**
4. 製作自己的 **興趣清單與課表**
5. 練習結構化內容呈現

👉 核心能力：**讓資訊變得有條理（列表 + 表格）**

---

## 🧩 一、建立 index.html（引導式）

本次不直接給完整程式碼，請依照提示一步一步完成：

### Step 1：建立基本架構

* 建立 `lab4/index.html`
* 複製之前學過的 HTML 基本骨架
* 輸入 `<strong>!</strong>`，再按下 Tab 鍵，會自動產生基本架構
* 在 `<head>` 中加入 `<title>`（例如：我的生活）

---

### Step 2：加入標題

在 `<body>` 中加入：

* 主標題：例如「我的生活」

👉 提示：使用 `h1`

---

### Step 3：建立興趣清單（無序清單）

👉 先加入一個小標題（例如：我的興趣）
👉 提示：使用 `h2`

建立一個興趣列表，例如：

* 打球
* 聽音樂
* 看電影

👉 提示：

* 使用 `ul`（無序清單）
* 每一項使用 `li`

---

### Step 4：建立另一種清單（有序清單）

👉 先加入一個小標題（例如：每日任務）
👉 提示：使用 `h2`

建立一個「學習步驟」或「每日任務」：

👉 提示：

* 使用 `ol`
* 搭配 `li`

---

### Step 5：建立課表（表格）

👉 先加入一個小標題（例如：我的課表）
👉 提示：使用 `h2`

建立一個完整課表，例如：

| 節次  | 時間    | 星期一 | 星期二 | 星期三 |
| --- | ----- | --- | --- | --- |
| 第一節 | 08:10 | 數學  | 英文  | 國文  |
| 第二節 | 09:10 | 物理  | 化學  | 歷史  |
| 第三節 | 10:10 | 體育  | 生物  | 地理  |

👉 提示：

* 使用 `table`
* 每一列使用 `tr`
* 標題使用 `th`
* 內容使用 `td`

---

### Step 6：加入回首頁

👉 提示：

* 使用 `a`
* 路徑：`../index.html`

---

完成以上步驟，你就會完成自己的 lab4/index.html 🎉

## 🧠 二、HTML 新觀念

### 1️⃣ 清單（List）

#### 無序清單

```html
<ul>
  <li>項目</li>
</ul>
```

👉 項目沒有順序（●）

---

#### 有序清單

```html
<ol>
  <li>步驟</li>
</ol>
```

👉 會有編號（1. 2. 3.）

---

### 2️⃣ 表格（Table）

```html
<table>
  <tr>
    <th>標題</th>
    <th>標題</th>
  </tr>
  <tr>
    <td>內容</td>
    <td>內容</td>
  </tr>
</table>
```

📌 重點：

* table → 整張表
* tr → 一列
* th → 標題格
* td → 內容格

## ⚠️ 四、常見錯誤

❌ 清單沒顯示點點

👉 忘記使用 `ul` 或 `ol`

---

❌ 表格亂掉

👉 檢查：

* tr 是否包住 td
* 結構是否正確

---

❌ 忘記關標籤

```html
<li>內容
```

👉 少了 `</li>`

---

請記住三件事：

1. ul / ol 用來做清單
2. li 是清單項目
3. table 用來做表格

## 🎨 五、加入 style.css 美化

建立：

```
lab4/style.css
```

```css
body {
    font-family: Arial, sans-serif;
    background-color: #f4f6f8;
    text-align: center;
    padding: 20px;
}

ul, ol {
    display: inline-block;
    text-align: left;
    margin: 20px auto;
}

table {
    margin: 20px auto;
    border-collapse: collapse;
}

th, td {
    border: 1px solid #ccc;
    padding: 10px 15px;
}

th {
    background-color: #3498db;
    color: white;
}
```

---

### 📌 提醒

記得在 HTML 加入：

```html
<link rel="stylesheet" href="style.css">
```
