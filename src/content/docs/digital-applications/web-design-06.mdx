# 🌐 Lab5：CSS 基礎認識與練習

---

## 🎯 本次學習目標

本次重點是「理解 CSS 是什麼」，並實際操作：

1. 了解 **什麼是 CSS（負責外觀）**
2. 認識 **CSS 的基本語法（選擇器、屬性、值）**
3. 理解 **HTML 與 CSS 的分工**
4. 使用提供的 HTML 練習 CSS
5. 完成 **至少 3 種樣式變化**

👉 核心能力：**知道 CSS 在做什麼，而不是只會複製貼上**

---

## 🧩 一、什麼是 CSS（重要觀念）

👉 HTML：負責內容（文字、圖片、結構）
👉 CSS：負責外觀（顏色、大小、排版）

📌 可以這樣想：

* HTML = 骨架
* CSS = 外觀

---

請使用以下 `lab5/index.html`：

```html
<!DOCTYPE html>
```

```
lab5/style.css
```

---

### Step 2：開始改畫面

現在開始思考：

👉 我要改哪個 HTML？（h1 / p / body / ul）

例如：

```css
h1 {
  color: blue;
}
```

👉 意思：
把畫面上的 **h1 變藍色**

---

再試更多：

```css
body {
  background-color: #f5f7fa;
  text-align: center;
}

p {
  font-size: 18px;
}
```

👉 核心概念：

**CSS = 選一個 HTML → 改它的樣子**

---

## 🧠 四、CSS 基本語法

```css
h1 {
  color: blue;
}
```

📌 結構說明：

* h1 → 選擇器（選到誰）
* color → 屬性
* blue → 值

## 🚀 九、加入一系列的美化這個 HTML 檔

請不要改 `index.html`，只修改 `style.css`，依照下面順序一步一步美化畫面。

---

### Step 1：先美化整個頁面

先選到 `body`，讓整體畫面不要太單調：

* 加入背景顏色
* 設定字體
* 讓內容置中
* 加入內距（padding）

例如：

```css
body {
  background-color: #f5f7fa;
  font-family: Arial, sans-serif;
  text-align: center;
  padding: 20px;
}
```

---

### Step 2：美化主標題

再選到 `h1`，讓標題更明顯：

* 修改文字顏色
* 放大字體

例如：

```css
h1 {
  color: #2c3e50;
  font-size: 32px;
}
```

---

### Step 3：美化段落文字

再選到 `p`，讓內文更容易閱讀：

* 調整文字大小
* 改變文字顏色

例如：

```css
p {
  font-size: 18px;
  color: #555;
}
```

---

### Step 4：美化清單

再選到 `ul`，讓興趣清單更整齊：

* 讓清單不要太靠左
* 調整對齊方式

例如：

```css
ul {
  display: inline-block;
  text-align: left;
}
```

---

### Step 5：美化表格

再選到 `table`、`th`、`td`，讓表格更清楚：

* 表格置中
* 加上框線
* 加入內距
* 幫標題列加底色

例如：

```css
table {
  margin: 20px auto;
  border-collapse: collapse;
}

th,
td {
  border: 1px solid #ccc;
  padding: 10px 15px;
}

th {
  background-color: #3498db;
  color: white;
}
```

---

### Step 6：練習 id 與 class

這個 HTML 裡有：

* `id="my-table"`
* `class="row1"`
* `class="row2"`

你可以試著分別改它們的樣式：

```css
#my-table {
  background-color: white;
}

.row1 {
  background-color: #f9f9f9;
}

.row2 {
  background-color: #eef6ff;
}
```

---

### Step 7：完成至少 3 種以上變化

請至少完成下面其中 3 種：

1. 改背景顏色
2. 改標題顏色
3. 改文字大小
4. 讓內容置中
5. 美化表格
6. 練習 id / class
