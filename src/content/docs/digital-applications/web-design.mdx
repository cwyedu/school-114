這是簡單的、使用AI輔助的網頁設計非常入門介紹，預計10次上課：

| 週次   | 主題                   | 教學重點                                                 | Lab 作業內容                     | Lab 目標         |
| ---- | -------------------- | ---------------------------------------------------- | ---------------------------- | -------------- |
| 第1週  | [Lab0：網站架構與 GitHub](/digital-applications/web-design-01) | 檔案結構、index.html、資料夾概念、GitHub Repository、GitHub Pages | 建立個人網站資料夾與首頁，設定 GitHub Pages | 成功看到自己的網站網址    |
| 第2週  | [Lab1：AI 抽籤網頁](/digital-applications/web-design-02)         | 使用 AI 產生程式碼、修改內容、體驗互動效果                              | 製作可點擊抽籤的網頁                   | 按鈕可運作，內容可自行修改  |
| 第3週  | Lab2：自我介紹頁面          | HTML 基本結構、h1~h3、p、br、hr                              | 建立自我介紹頁面並連回首頁                | 正確使用基本文字標籤     |
| 第4週  | Lab3：圖片與超連結頁面        | img、a、相對路徑、images 資料夾                                | 建立興趣介紹頁，加入圖片與外部連結            | 能正確顯示圖片與連結     |
| 第5週  | Lab4：清單與表格頁面         | ul、ol、li、table、tr、td、th                              | 製作興趣清單與課表                    | 完成清單與3x3以上表格   |
| 第6週  | Lab5：CSS 基礎美化        | style、color、background-color、font-size、text-align    | 美化前面任一 Lab 頁面                | 至少3種樣式變化       |
| 第7週  | Lab6：個人卡片設計          | div、margin、padding、border、width                      | 製作個人介紹卡片                     | 有邊框與間距設定       |
| 第8週  | Lab7：Flex 排版設計       | display:flex、justify-content                         | 製作三張卡片並排頁面                   | 成功使用 flex 排版   |
| 第9週  | Lab8：網站整合與導覽列        | 多頁串接、導覽列設計、統一風格                                      | 整合所有 Lab 頁面                  | 每頁可互相連結、風格一致   |
| 第10週 | Lab9：期末成果網站          | 排版優化、網站檢查、成果展示                                       | 完成並發表個人網站                    | 至少5頁、無壞連結、成功上線 |
