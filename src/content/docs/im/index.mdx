這是 114 學年，互動多媒體課程的資料與記錄

## 內容

既然是**互動**多媒體，我就在想，互動性在哪？VR、AR 沒有工具，沒法做太多，不如就從一些簡單的互動做起。下面有一些項目，不如就在 AI 的輔助下，挑一些來完成：

- 上學年送舊時有玩**猜歌遊戲**，就寫個猜歌程式
- 雖然不是直接互動，但**Line 貼圖**是一個不錯的項目
- **互動故事**：互動任務遊戲故事、互動式校園導覽…
- 鏡頭互動：
  - 用 **WebAR** 做一個掃杯墊就跳出 AR
  - **臉部濾鏡遊戲**
    - 工具：p5.js + ml5.js (臉部偵測)
    - 玩法：
      - 鏡頭偵測臉部 → 疊上兔耳朵、墨鏡、貓鬍鬚
      - 表情互動：張嘴會播放「喵~」聲
  - **手勢控制的 Flappy Bird**
    - 工具：MediaPipe Hands
    - 玩法：
      - 鏡頭偵測手掌高度
      - 左手抬高 → 小鳥飛起
  - **動作體感遊戲（校園健身版）**
    - 工具：PoseNet (ml5.js)
    - 玩法：
      - 鏡頭偵測全身骨架
      - 偵測「跳躍」次數 → 記分
      - 學生比賽：誰跳最多分數最高
  - **互動式校園吉祥物 AR**
    - 工具：MindAR + A-Frame
    - 玩法：
      - 拿手機掃校徽 → 校徽變 3D 吉祥物
      - 吉祥物可以跟隨手指移動
      - 也能對臉部表情做反應
