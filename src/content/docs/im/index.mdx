### 內容

既然是**互動**多媒體，我就在想，互動性在哪？VR、AR 沒有工具，沒法做太多，不如就從一些簡單的互動做起。下面有一些項目，不如就在 AI 的輔助下，挑一些來完成：

<Collapsible title="專案清單">
- 上學年送舊時有玩**猜歌遊戲**，就寫個猜歌程式
- 雖然不是直接互動，但**Line 貼圖**是一個不錯的項目
- **互動故事**：互動任務遊戲故事、互動式校園導覽…
- 鏡頭互動：
  - 用 **WebAR** 做一個掃杯墊就跳出 AR
  - **臉部濾鏡遊戲**
    - 工具：p5.js + ml5.js (臉部偵測)
    - 玩法：
      - 鏡頭偵測臉部 → 疊上兔耳朵、墨鏡、貓鬍鬚
      - 表情互動：張嘴會播放「喵~」聲
  - **手勢控制的 Flappy Bird**
    - 工具：MediaPipe Hands
    - 玩法：
      - 鏡頭偵測手掌高度
      - 左手抬高 → 小鳥飛起
  - **動作體感遊戲（校園健身版）**
    - 工具：PoseNet (ml5.js)
    - 玩法：
      - 鏡頭偵測全身骨架
      - 偵測「跳躍」次數 → 記分
      - 學生比賽：誰跳最多分數最高
  - **互動式校園吉祥物 AR**
    - 工具：MindAR + A-Frame
    - 玩法：
      - 拿手機掃校徽 → 校徽變 3D 吉祥物
      - 吉祥物可以跟隨手指移動
      - 也能對臉部表情做反應
</Collapsible>

【內容簡述】

本作品記錄我在「互動多媒體實務」課程中，結合資料處理科的邏輯思維與數位內容製作的學習成果。我開發了「音樂猜謎遊戲」，實踐前端網頁邏輯判斷與資源管理；並製作「SDGS AR 體驗」，探索擴增實境技術在永續議題上的應用。透過這兩個專題，我不僅掌握了互動技術，更學會如何將靜態資料轉化為沈浸式體驗，展現資訊整合與問題解決的專業能力。

一、 學習背景與動機

在資訊爆炸的時代，單向的資訊傳遞已難以吸引大眾關注。本學期在「互動多媒體實務」課程中，我希望挑戰將「靜態資料」轉化為「動態體驗」。我一共開發了兩個不同性質的專題：

音樂猜謎遊戲：旨在練習前端網頁的判斷邏輯與多媒體資源調用，強化程式開發基礎。

SDGS AR 體驗：結合聯合國永續發展目標，嘗試利用新興的擴增實境（AR）技術，提升社會議題的傳播趣味性與互動深度。

二、 實際行動與操作方法

1. 音樂猜謎系統 (Web-Based Interaction)

介面開發：使用 HTML/CSS 建立具備回應式設計（Responsive Web Design）的遊戲介面，確保作品在桌機與行動裝置上皆能獲得良好體驗。

邏輯設計：運用 JavaScript 撰寫核心控制腳本，包含音訊播放/暫停觸發，並透過 if-else 邏輯運算處理使用者輸入，比對資料庫答案之正確性。

資源管理：實踐資處科所學之資料預處理概念，針對音檔進行轉碼與壓縮優化，確保在維持音質的前提下大幅提升網頁載入效率。

2. SDGS AR 體驗 (Augmented Reality)

議題分析與素材開發：深入研究 SDGS 指標（如：海洋生態或氣候行動），將抽象資訊轉化為具象的 3D 模型與動態辨識圖卡，進行視覺化呈現。

AR 互動實作：運用 AR 編輯工具將數位內容與現實場景串接。在設計辨識圖時，需考量圖像對比度與特徵點（Feature Points）分佈，以提升追蹤穩定度。

三、 所培養的能力與知識

資料結構化與邏輯思考：在開發猜謎遊戲時，我理解到如何系統化地整理題目與答案的對應關係，而非單純處理零碎檔案。

多媒體資源整合能力：成功串接影像、音訊與互動腳本，並理解使用者反饋（Feedback）對於系統即時狀態更新的重要性。

數位內容策展與傳達：學會篩選關鍵資訊，並利用互動科技將枯燥的環境議題轉化為具吸引力的數位教材。

四、 問題、困難與反思

困難挑戰：在開發音樂猜謎時，曾面臨網頁非同步載入導致音訊延遲播放的問題，造成使用者操作時產生「反應遲鈍」的錯覺。

解決方式：我透過查閱網路技術論壇與向老師請益，學習到預載（Preload）與事件監聽器（Event Listener）的應用技巧，優化了腳本執行順序。

自我反思：製作過程中我意識到「使用者體驗 (UX)」的重要性。開發者認為直覺的流程，使用者未必能理解。這讓我未來在設計資訊系統時，會更重視介面的易用性（Usability）測試。

五、 未來學習與升學連結

這學期的專題製作讓我對「互動科技」與「資訊管理」產生了濃厚興趣。在未來升學路上，我希望能在 管理資訊系統 (MIS) 領域深耕。我計畫將資處科習得的後台管理能力與本課程的多媒體技術結合，未來目標是開發出具備雲端資料庫支援、可分析使用者行為的大型互動教育平台。
