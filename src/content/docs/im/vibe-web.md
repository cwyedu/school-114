## 一、課程說明

在過去的學習中，我們多半使用「網頁版 AI 助理」來協助撰寫網頁或程式碼。然而，這種方式在實作上有一些不便之處，例如：

* 程式碼不易即時測試
* 修改與除錯流程較繁瑣
* 缺乏整合開發環境的支援

因此，本課程將改用一個更適合開發的工具：
👉 [https://antigravity.google/](https://antigravity.google/)

透過這個工具，我們將實際動手製作 3 個簡單的網頁小遊戲。

---

## 二、網站結構

本課程的網站專案結構如下：

```
imweb/
├── index.html   ← 主頁（入口）
├── lab1/
│   └── index.html   ← 猜歌遊戲
├── lab2/
│   └── index.html   ← 照相亭
└── lab3/
    └── index.html   ← 自己發想
```

說明：

* **imweb**：專案根目錄
* **lab1**：放置「猜歌遊戲」相關檔案
* **lab2**：放置「照相亭」相關檔案
* **lab3**：放置「自行發想遊戲」相關檔案

## 三、開發工具

### Google Antigravity

Google Antigravity 是一個線上開發環境，可以讓你：

* 即時撰寫與執行程式碼
* 快速查看成果
* 方便進行修改與測試

### GitHub Desktop

GitHub Desktop 是一個圖形化版本控制工具，可以讓你：

* 管理專案版本（版本控制）
* 將專案上傳到 GitHub
* 查看與回復歷史修改紀錄
* 方便進行多人協作

## 四、教學流程

以下為本課程的整體教學順序：

1. 使用 Antigravity 建立上述檔案架構，產生基本測試網頁

3. lab2：照相亭
4. lab1：猜歌遊戲
5. lab3：自行發想的遊戲
