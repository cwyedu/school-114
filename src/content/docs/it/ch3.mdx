```
請扮演一位專業的「全端互動設計工程師」。你的任務是創建一個單一的 HTML 檔案（包含所有必要的 CSS 和 JavaScript 程式碼），以實現一個視覺化展示「二分搜尋（Binary Search）」原理的猜數字遊戲。

**遊戲規格與功能要求 (Game Specification):**
1.  **目標範圍：** 猜測的數字範圍是 1 到 100。
2.  **檔案結構：** 必須是單一的 HTML 檔案，所有 `<style>` (CSS) 和 `<script>` (JavaScript) 內容必須內嵌於 `<head>` 或 `<body>` 標籤中。
3.  **初始化顯示：** 網頁載入時，必須清晰地列出 1 到 100 的所有數字。每個數字應被單獨包裝（例如用 `<div>` 或 `<span>`），以便後續的 CSS 樣式控制。
4.  **遊戲流程：**
    a.  系統在後台隨機生成一個 1-100 的目標數字。
    b.  提供一個輸入框和「猜測」按鈕，讓使用者輸入他們的猜測。
    c.  每次猜測後，系統根據使用者的輸入，判斷是「太大」、「太小」還是「猜中」。
5.  **二分搜尋視覺化 (Binary Search Visualization):**
    a.  **核心要求：** 每次使用者猜測後，系統必須更新視覺化，將所有**已被排除**的數字（即不再可能成為答案的數字）應用**變暗 (e.g., opacity: 0.3 或 background-color: #eee)** 的樣式，以模擬二分搜尋縮小範圍的過程。
    b.  系統必須維護一個當前**有效搜尋範圍**的上下限 (`low` 和 `high` 變數)，並用這些變數來控制數字的視覺化狀態。
6.  **結束條件：** 猜中後，顯示祝賀訊息（"Bingo! You got it!"），並明確展示使用者總共猜測了**多少次**。
7.  **CSS 樣式：** 樣式應簡潔且利於視覺化。使用網格（Grid）或彈性盒（Flexbox）來排列 100 個數字，確保排版整齊。

**Output Format:**
僅輸出完整的單一 HTML 檔案程式碼，無需任何解釋或額外文字。
```

<Collapsible title="猜生日魔術互動應用">

```
請扮演一位「互動式教育程式設計師」。你的任務是創建一個單一的 HTML 檔案（包含所有必要的 CSS 和 JavaScript 程式碼），以實現一個用於「二進位猜生日魔術」的互動式網頁應用程式。

**應用規格與功能要求 (Application Specification):**
1.  **檔案結構：** 必須是單一的 HTML 檔案，所有 `<style>` (CSS) 和 `<script>` (JavaScript) 內容必須內嵌於 `<head>` 或 `<body>` 標籤中。
2.  **範圍設定：** 遊戲範圍鎖定在 **1 到 31**，對應 5 張卡片。
3.  **卡片設計與呈現：**
    a.  網頁必須清晰地展示 5 張卡片，每張卡片應包含**所有**包含該卡片起始數字的數字（1-31）。
    b.  **卡片的起始數字（即 $2^n$）：** 1, 2, 4, 8, 16。
    c.  每張卡片應設計為**可點擊**或搭配複選框，讓使用者標記「生日數字是否有出現在這張卡片上」。
    d.  卡片設計應簡潔且利於閱讀（例如：使用 CSS Grid 排列卡片上的數字）。
4.  **互動流程：**
    a.  使用者完成 5 張卡片的選擇後，點擊一個「猜測我的生日」按鈕。
    b.  JavaScript 程式碼必須將所有被選中卡片的**起始數字**加總起來，得出答案。
5.  **原理與結果展示 (Step-by-Step Explanation):**
    在猜測結果出來後，必須在網頁上新增一個「原理分析區」，**步驟化**地展示魔術的原理：
    * **步驟 1：卡片加總。** 顯示所有被選中卡片的起始數字，以及它們的總和。
        * *範例輸出：* 「您選擇了起始數字為 $1, 4, 16$ 的卡片。總和是 $1 + 4 + 16 = 21$。」
    * **步驟 2：二進位分解。** 解釋該總和如何對應到二進位位元。
        * *範例輸出：* 「這總和 $21$ 其實就是 $16 (2^4) + 4 (2^2) + 1 (2^0)$，這證明了 $21$ 的二進位表示是 $10101_2$。」
    * **步驟 3：最終答案。** 顯示最終猜測的生日數字。

**Output Format:**
僅輸出完整的單一 HTML 檔案程式碼，無需任何解釋或額外文字。
```
