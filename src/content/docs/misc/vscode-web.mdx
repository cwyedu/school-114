## 使用 VS Code

### 安裝 Extension（擴充套件）

1. 下載安裝 [Visual Studio Code](https://code.visualstudio.com/)
1. 開啟
2. 點擊左側【Extensions（擴充套件）】圖示。
3. 在搜尋欄輸入 `Chinese`。
4. 點擊 **Chinese (Traditional) Language Pack for Visual Studio Code** →【Install】。
5. 依提示重新整理或重新載入 VS Code。
6. 在搜尋欄輸入 `Live Server`。
7. 點擊作者為 **Ritwick Dey** 的 Live Server →【Install】。
8. 安裝完成後，在 HTML 檔案上按右鍵，確認出現【Open with Live Server】。

## 網頁的基本組成

一般的網頁通常由以下三種檔案組成：

1. **HTML**： 負責網頁的基本結構與內容（例如：文字、圖片、按鈕）。
2. **CSS**： 負責網頁的外觀與樣式（例如：顏色、字型、版面配置）。
3. **JavaScript**： 負責網頁的互動與功能（例如：按鈕點擊、動態效果）。
