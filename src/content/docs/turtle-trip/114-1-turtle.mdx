## 第 5 週 模式識別

![](https://i.imgur.com/sJvqMzg.gif)

有沒有發現，這 2 張圖都可以找出相似的模式嗎，我們可以把相似的部分抽出來，變成一個函數 (function)，然後重覆使用這個函數來畫出整張圖。

<Collapsible title="咒語">

左邊的圖：它可以折解成**一支像箭頭的圖**，然後轉一圈重覆畫 60 次。

```
先做一個函數：往前走一大段距離，左轉(b)，往前(a)，退回(a)，右轉(b)，往前(a)，退回(a)，右轉(b)，往前(a)，退回(a)，左轉(b)，退回一大段到原來的位置。

然後重覆畫 60 次，每次右轉 6 度
```

這裡最有可能發生的錯誤是：

- 方向轉錯 (最前面的三個小線段畫完後，方向沒轉回來)
- 距離錯
- 函數沒有正確回到原來的位置

右邊的圖：

```
先做一個函數：畫一個三角形並填滿顏色，再畫一個小點的三角形並填滿顏色，再畫個更小的三角形並填滿顏色。

然後重覆畫 5 次，每次右轉 72 度
```

</Collapsible>

## 第 6 週 時鐘

![](https://i.imgur.com/i7rA4rH.gif)

<Collapsible title="咒語">

先畫出刻度

```
先做一個函數(畫刻度)：提筆，走一段長距度，畫一小段，退回(長+短)距確回到原來位置。可以設定參數：長距離、短距離、短距離的寬度

畫刻度：重覆畫 60 次，每次右轉 6 度。逢5的倍數，畫長一點，寬一點。
```

再畫框

```
先做一個函數(畫刻度)：提筆，走一段長距度，畫一小段，退回(長+短)距確回到原來位置。可以設定參數：長距離、短距離、短距離的寬度

畫刻度：重覆畫 60 次，每次右轉 6 度。逢5的倍數，畫長一點，寬一點。

做一個畫框的函數：可以設定：半徑、框的粗細、框的顏色
畫框：用畫框的函數，畫2個，內框粗一點，接聽一個細一點的外框
```

再畫出指針

```
先做一個函數(畫刻度)：提筆，走一段長距度，畫一小段，退回(長+短)距確回到原來位置。可以設定參數：長距離、短距離、短距離的寬度

畫刻度：重覆畫 60 次，每次右轉 6 度。逢5的倍數，畫長一點，寬一點。

做一個畫框的函數：可以設定：半徑、框的粗細、框的顏色
畫框：用畫框的函數，畫2個，內框粗一點，接聽一個細一點的外框

做一個畫指針的函數，可以設定：長度、寬度、顏色、角度
畫指針：用畫指針的函數，畫 3 個指針，時針、分針、秒針，時間設定在3點0分40秒
```

最後，你自己去調顏色吧！

</Collapsible>
